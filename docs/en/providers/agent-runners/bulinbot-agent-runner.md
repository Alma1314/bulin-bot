# Built-in Agent Runner

By default, NovaBot uses the built-in Agent Runner as the default executor. You don't need to configure anything to use NovaBot's powerful built-in Agent Runner.

![image](https://files.novabot.app/docs/source/images/nova-bot-agent-runner/image.png)

With the built-in Agent Runner, you can use NovaBot's [MCP Server](/use/mcp), [Knowledge Base](/use/knowledge-base), [Web Search](/use/websearch), and persona features.
